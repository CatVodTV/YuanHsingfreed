# 猫影视TV 2.1.1 BETA5 新功能
本地接口 clan://协议 简易设定流程：
* 升级猫影视TV至 2.1.1 Beta5
* 设置 --> 存储权限 --> 允许
* 下载底下的懒人包
* 将 懒人包 解压缩 至 "根目录"
* 设置 --> 当前接口配置
	* 直接黏贴底下的 配置接口设定
	* 或是也可以 透过扫码或浏览器访问网址 --> 设置配置接口
